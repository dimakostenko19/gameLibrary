# React game library
Этот проект представляет собой React-приложение, которое отображает список игр, разделенных по жанрам. Пользователи могут выбрать жанр, чтобы просмотреть игры, доступные в этой категории.

## Features
- Отображение списка игр по жанрам.
- Поиск и отображение игр по заданому жанру.
- Динамическая выборка данных с помощью [RAWG API](https://rawg.io/apidocs).
